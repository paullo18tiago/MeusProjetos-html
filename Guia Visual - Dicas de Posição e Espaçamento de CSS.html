🎨 Guia Visual Completo de CSS

📦 Display: Flex (Flexbox)

O que faz: Transforma o container em um sistema flexível para alinhar e distribuir itens.
Propriedades principais:
• display: flex - Ativa o Flexbox
• justify-content: center - Centraliza horizontalmente
• align-items: center - Centraliza verticalmente
• gap: 15px - Espaço entre os itens
• flex-direction: row - Direção (row ou column)

📍 Position (Posicionamento)

Tipos de position:
• static - Fluxo normal (padrão)
• relative - Move relativo à posição original
• absolute - Posição absoluta ao elemento pai
• fixed - Fixo na tela (não rola)
• sticky - Híbrido entre relative e fixed

📏 Padding e Margin (Espaçamento)

Padding: Espaço INTERNO (entre conteúdo e borda)
Margin: Espaço EXTERNO (entre elemento e vizinhos)

Atalhos:
• padding: 20px - Todos os lados = 20px
• padding: 10px 20px - Vertical=10px, Horizontal=20px
• padding: 10px 20px 15px 25px - Topo, Direita, Baixo, Esquerda
• padding-top: 10px - Apenas o topo
• margin: 0 auto - Centraliza horizontalmente
Esta caixa verde tem padding: 20px interno
A caixa vermelha externa também tem padding: 20px

🔲 Border e Border-Radius (Bordas)

Propriedades:
• border: 2px solid #000 - Espessura, estilo, cor
• border-radius: 10px - Cantos arredondados
• border-radius: 50% - Círculo perfeito
• border-top: 4px solid red - Apenas um lado

🌊 Overflow (Conteúdo Transbordante)

Controla o que acontece quando o conteúdo não cabe:
• overflow: visible - Transborda (padrão)
• overflow: hidden - Esconde o que passa
• overflow: auto - Barra de rolagem quando necessário
• overflow-y: scroll - Rolagem apenas vertical

Transforma elementos sem afetar o layout:
• transform: translateX(20px) - Move horizontalmente
• transform: translateY(-10px) - Move verticalmente
• transform: scale(1.2) - Aumenta/diminui
• transform: rotate(45deg) - Rotaciona

📐 Calc e Unidades Dinâmicas

Cálculos e unidades responsivas:
• calc(100% - 40px) - Cálculo dinâmico
• clamp(10px, 5%, 50px) - Mín, ideal, máx
• vh - Porcentagem da altura da tela
• vw - Porcentagem da largura da tela
• rem - Relativo à fonte raiz
• em - Relativo à fonte do elemento
• % - Porcentagem do elemento pai

🎭 Z-Index (Camadas/Profundidade)

Controla a ordem de empilhamento:
• Valores maiores = mais na frente
• z-index: 1 - Camada 1
• z-index: 999 - Muito na frente
• z-index: -1 - Atrás
• Só funciona com position diferente de static!

📦 Box-Sizing (Cálculo de Tamanho)

Como o tamanho é calculado:
• box-sizing: content-box - Padrão: padding e border SÃO SOMADOS ao width
• box-sizing: border-box - Melhor: padding e border INCLUÍDOS no width

Exemplo:
Com width: 200px e padding: 20px:
• content-box = 240px total (200 + 20 + 20)
• border-box = 200px total (padding incluído)

🎨 Display (Tipos de Exibição)

Tipos principais de display:
• display: block - Ocupa linha inteira (div, p, h1)
• display: inline - Lado a lado, não aceita width/height (span, a)
• display: inline-block - Lado a lado + aceita width/height
• display: flex - Container flexível
• display: grid - Sistema de grade
• display: none - Esconde completamente

🎯 Justify-Content e Align-Items (Alinhamento)

Usado com Flexbox:

• flex-start - Início

• space-around - Espaço ao redor

• flex-start - Topo
• center - Centro
• flex-end - Fundo

📱 Responsividade (Media Queries)

Adapta o layout para diferentes tamanhos de tela:
• @media (max-width: 768px) - Telas menores que 768px
• @media (min-width: 1024px) - Telas maiores que 1024px
• @media (orientation: landscape) - Modo paisagem

Breakpoints comuns:
• Mobile: até 768px
• Tablet: 768px - 1024px
• Desktop: acima de 1024px

🎓 Resumo dos Conceitos

Ordem de organização CSS recomendada:
elemento {
  /* 1. DISPLAY E POSIÇÃO */
  display: flex;
  position: relative;
  z-index: 10;

  /* 2. TAMANHO */
  width: 100%;
  max-width: 900px;
  min-height: 100vh;
  box-sizing: border-box;

  /* 3. ESPAÇAMENTO */
  margin: 0 auto;
  padding: 20px;
  gap: 15px;

  /* 4. ALINHAMENTO */
  justify-content: center;
  align-items: center;

  /* 5. VISUAL */
  background: #667eea;
  border: 2px solid #000;
  border-radius: 10px;
  overflow: hidden;

  /* 6. TRANSFORMAÇÃO */
  transform: translateY(10px);
  transition: all 0.3s;

  /* 7. TEXTO */
  font-family: Arial;
  color: white;
}
